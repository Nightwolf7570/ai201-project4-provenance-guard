# Provenance Guard

Provenance Guard is a Flask backend for CodePath AI201 Project 4. It analyzes submitted text for attribution signals, returns a confidence-aware transparency label, supports creator appeals, rate limits submissions, and records structured audit logs.

This README will be expanded after the planning and implementation milestones with architecture details, detection signal explanations, confidence scoring evidence, rate-limit evidence, audit-log samples, appeal handling, known limitations, and AI usage notes.
